# HPとブログのホスティングサービス

## 開発環境
PHP + Laravel
DB: MySQL

## 実装済機能
- ブログ
    - 投稿機能
        - 新規投稿
        - 記事編集
        - 入力内容のvalidate
        - WYSIWYGエディタ
    - 記事削除機能
        - ソフトデリート
    - 記事一覧表示機能
        - 同日の場合まとめて表示したい
    - DBテーブルのリレーション管理
		ORmapper
- コード
    - 作成したコードの一覧とその説明

## 追加したい機能
- 管理ユーザ登録機能
    - 記事を投稿できるユーザを登録できる
- 管理ユーザログイン機能
    - 名前とパスワード	
- ページネーション
- 各記事から、編集画面へのリンク
- 単体テスト機能
- 統合テスト機能
